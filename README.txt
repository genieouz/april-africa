April Africa Project
Décompressez le dossier april-africa dans votre serveur web. 
Importez La base de données qui se trouve dans le dossier april-africa/db dans une db appelé projet_april


Dans ce projet j'ai laissé la possibilité à l'utilisateur de créer un compte avec son role. En pratique, l'utilisateur ne devra avoir que la possibilité de se connecter et non pas de s'inscrire. Une fois connecté, s'il est admin, alors il pourra créer des utilisateur en définissant leur role. Les roles que vous devez saisir lors de la création d'un utilisateur sont admin, commercial, consultant. Veuillez respecter la casse.
